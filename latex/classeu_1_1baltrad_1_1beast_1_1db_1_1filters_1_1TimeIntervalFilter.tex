\doxysubsection{Detailed Description}
Fetches all objects that are within a specified start -\/ stop date/time interval. 

You can also ignore specifying start and/or stop date/time but if you ignore both you probably want to use a different filter.

The filter compares\+: object equality (REQUIRED) source node id equality (OPTIONAL) start date-\/time \texorpdfstring{$<$}{<}= requested \texorpdfstring{$<$}{<} stop date-\/time

It is also possible to limit the number of returned values by using the limit. Default is unlimited (0). The limiting function is ordered by descending date/time which means that if you specify limit 1 it will be the most recent time within the time interval.
