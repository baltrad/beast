\doxysubsection{Detailed Description}
The publisher (manager) that keeps track of publishing file requests to different nodes. 

There is one thread pool executor / node so that a slowly responding node doesn\textquotesingle{}t lock everything for the other nodes. \begin{DoxyAuthor}{Author}
