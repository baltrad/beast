\doxysubsection{Detailed Description}
Upload files using SCP. 

Uses $\sim$/.ssh/known\+\_\+hosts and $\sim$/.ssh/known\+\_\+hosts2 for host verification and $\sim$/.ssh/id\+\_\+rsa and $\sim$/.ssh/id\+\_\+dsa for keys.

More thorough configuration could be provided through query arguments in the future.
