\doxysubsection{Detailed Description}
The scheduler within beast. 

The scheduler works like any other IBlt\+Message producer and sends Blt\+Scheduled\+Job\+Message\textquotesingle{}s. This means that in order to handle scheduled jobs the appropriate rule must be supporting this.

You ought to specify job\+Name (rule name) in order to get the correct rule executed instead of the ordinary broadcast but that is up to you.
